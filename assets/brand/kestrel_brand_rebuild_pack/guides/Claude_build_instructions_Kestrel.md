# Claude build instructions, Kestrel marketing pack

## Objective
Use the supplied approved Kestrel logo as the source of truth and generate consistent collateral across email, PowerPoint, PDF covers, banners and tiles.

## Inputs
- `kestrel_logo_original.png`
- `brand_message_original.png`

## Design principles
- premium, strategic, modern
- avoid militarised clichés
- use breathing room
- retain the approved logo lock-up without distortion
- build support graphics from the envelope / chevron geometry, not unrelated motifs

## Palette
Use the hex values listed in `Kestrel_brand_rebuild_guide.md`.

## Asset rules
1. Use the light transparent logo on light backgrounds.
2. Use the supplied dark-mode transparent logo on dark backgrounds.
3. For icons, use the symbol-only files.
4. Keep the descriptor `DEFENCE INSIGHTS` intact for the master brand.
5. If a campaign sub-line is needed, place it outside the core lock-up.

## Layout rules
- Left-align the logo where practical.
- Use bold headlines with clean sans serif typography.
- Keep blue/violet accents to 10 to 20 percent of the layout.
- Avoid heavy gradients and shadows.

## Preferred output set
- email masthead light and dark
- social tile 1200x627
- square social tile 1080x1080
- PowerPoint cover 16:9
- section divider 16:3
- Teams / SharePoint banner
- PDF/A4 cover

## Sample brand line
Minimising future grad/hiring challenge, low-cost brand play that builds visibility and future talent pipeline
